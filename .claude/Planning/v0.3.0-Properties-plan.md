> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship the complete data layer for the 11-type Property catalog, the SQLite indexer (pulled forward from v0.3.3), and placeholder UI for every interaction so Properties behavior is verifiable end-to-end across Pages, Items, AgendaTask, and AgendaEvent.

**Architecture:** Eleven phases (A–K), each green standalone via the stub-and-progressively-replace paradigm (`// Guidelines//Paradigm-Decisions.md` decision #4). Phase B (`SchemaTransaction`) is the load-bearing atomic-write primitive used by C/D/G/H. SQLite indexer (Phase E) publishes from every manager mutation post-commit. ID-truth model enforced everywhere: property identity = stable ULID, name is renameable display.

**Tech Stack:** Swift 6 (strict concurrency + ExistentialAny), SwiftUI, AppKit interop (NSTextView, NSSplitView), GRDB.swift (SQLite via SPM), TextKit 2 + swift-markdown for editor (existing), Apple ULID generation pattern (existing).

```swift
public enum RelationScope: Codable, Hashable, Sendable {
    case pageType(String)
    case itemType(String)
    case pageCollection(String)
    case itemCollection(String)
    case contextTier(Int)

    enum CodingKeys: String, CodingKey {
        case kind, pageTypeID = "page_type_id", itemTypeID = "item_type_id"
        case pageCollectionID = "page_collection_id", itemCollectionID = "item_collection_id"
        case tier
    }

    public init(from decoder: any Decoder) throws {
        let c = try decoder.container(keyedBy: CodingKeys.self)
        let kind = try c.decode(String.self, forKey: .kind)
        switch kind {
        case "page_type":       self = .pageType(try c.decode(String.self, forKey: .pageTypeID))
        case "item_type":       self = .itemType(try c.decode(String.self, forKey: .itemTypeID))
        case "page_collection": self = .pageCollection(try c.decode(String.self, forKey: .pageCollectionID))
        case "item_collection": self = .itemCollection(try c.decode(String.self, forKey: .itemCollectionID))
        case "context_tier":    self = .contextTier(try c.decode(Int.self, forKey: .tier))
        default: throw DecodingError.dataCorruptedError(forKey: .kind, in: c, debugDescription: "Unknown kind: \(kind)")
        }
    }

    public func encode(to encoder: any Encoder) throws {
        var c = encoder.container(keyedBy: CodingKeys.self)
        switch self {
        case .pageType(let id):       try c.encode("page_type", forKey: .kind);       try c.encode(id, forKey: .pageTypeID)
        case .itemType(let id):       try c.encode("item_type", forKey: .kind);       try c.encode(id, forKey: .itemTypeID)
        case .pageCollection(let id): try c.encode("page_collection", forKey: .kind); try c.encode(id, forKey: .pageCollectionID)
        case .itemCollection(let id): try c.encode("item_collection", forKey: .kind); try c.encode(id, forKey: .itemCollectionID)
        case .contextTier(let t):     try c.encode("context_tier", forKey: .kind);    try c.encode(t, forKey: .tier)
        }
    }
}
```

```swift
import Foundation

public enum SchemaTransactionError: Error {
    case stageFailed(URL, any Error)
    case commitFailed(URL, any Error)
}

public final class SchemaTransaction {
    private struct StagedWrite {
        let payload: Data
        let target: URL
        let temp: URL
    }
    private var pending: [StagedWrite] = []

    public init() {}

    public func stage(payload: Data, to url: URL) {
        let temp = url.appendingPathExtension("txn-\(ULID.generate())")
        pending.append(StagedWrite(payload: payload, target: url, temp: temp))
    }

    /// Two-phase commit:
    ///   1. Clean stale `*.txn-*` siblings in target directories (idempotence).
    ///   2. Write all payloads to temp paths. If ANY fails: delete already-staged temps + throw.
    ///   3. Atomic rename all temps to targets. If ANY rename fails: best-effort restore prior targets.
    public func commit() throws {
        cleanStaleTemps()

        // Phase 1: stage all temps.
        do {
            for write in pending {
                try write.payload.write(to: write.temp, options: [.atomic])
            }
        } catch {
            pending.forEach { try? FileManager.default.removeItem(at: $0.temp) }
            throw SchemaTransactionError.stageFailed(pending.last?.target ?? URL(fileURLWithPath: "/"), error)
        }

        // Phase 2: rename all to targets in deterministic order.
        var renamed: [(URL, URL)] = []  // (target, backup-of-prior)
        for write in pending {
            do {
                let backup = write.target.appendingPathExtension("bak-\(ULID.generate())")
                if FileManager.default.fileExists(atPath: write.target.path) {
                    try FileManager.default.moveItem(at: write.target, to: backup)
                    renamed.append((write.target, backup))
                }
                try FileManager.default.moveItem(at: write.temp, to: write.target)
            } catch {
                // Rollback: restore backups, delete partial writes.
                for (target, backup) in renamed.reversed() {
                    try? FileManager.default.removeItem(at: target)
                    try? FileManager.default.moveItem(at: backup, to: target)
                }
                pending.forEach { try? FileManager.default.removeItem(at: $0.temp) }
                throw SchemaTransactionError.commitFailed(write.target, error)
            }
        }

        // Phase 3: delete backups (success).
        for (_, backup) in renamed {
            try? FileManager.default.removeItem(at: backup)
        }
    }

    private func cleanStaleTemps() {
        let dirs = Set(pending.map { $0.target.deletingLastPathComponent() })
        for dir in dirs {
            guard let contents = try? FileManager.default.contentsOfDirectory(atPath: dir.path) else { continue }
            for entry in contents where entry.contains(".txn-") {
                try? FileManager.default.removeItem(at: dir.appendingPathComponent(entry))
            }
        }
    }
}
```

## Cross-cutting concerns

- **Atomic-write discipline:** every schema mutation routes through `SchemaTransaction` (Phase B). Single-file updates stay on `AtomicJSON.write`.
- **Index update post-commit:** every manager mutation publishes to `IndexUpdater` after disk write. Order: disk-first, index-after. Index failure → log + queue for rebuild, never block user.
- **Concurrent-edit isolation:** schema sheet uses single-instance presentation guard (per-window); value edits unrestricted (live-save).
- **Parallel-session caveat (CLAUDE.md quirk #11):** never bundle Nathan's parallel-session edits (`External/MarkdownEngine/*`, `ContentView.swift`, `PageEditorView.swift`) into Properties-scope commits.
- **Swift 6 strict concurrency (quirk #5):** new managers follow `@MainActor @escaping () -> NexusContext` closure pattern; new types use `init(from decoder: any Decoder)` / `func encode(to encoder: any Encoder)`; error vars typed `(any Error)?`.
- **Test filter form (quirk #1):** `-only-testing:PommoraTests/<FilenameWithTests>` — FILENAME form. Verify count visually.
- **Build verification (quirk #3):** trust `xcodebuild`, not SourceKit squiggles. Builder subagent for all `xcodebuild` calls.
- **pbxproj auto-reordering (quirk #7):** revert incidental SymbolPicker/Yams/GRDB entries before commit.
- **`swift format` (quirk #12):** invoked as subcommand `swift format format --in-place ...` / `swift format lint --strict --recursive ...`. CI uses same.
