#### Lessons Learned

> **THE quirk to remember (Nathan, emphasized): why these bugs were slow to pin down ‚ÄĒ and the rules that kill each.** The meta-pattern across every Connections bug this session: **a UI symptom looked identical no matter which of four layers was broken.** A muted `[[link]]` could mean (a) not in the index, (b) resolver got the wrong string, (c) resolver worked but wasn't wired in, or (d) a stale compiled package ran old code. One symptom, four layers ‚Üí effort kept landing at the wrong one. Quirk #17 ("confirm the data before blaming it") exists *because this already happened* ‚ÄĒ and it recurred. The discipline: **instrument to name the failing layer before fixing; never guess-fix a chain by its end symptom.**

- **Stale local SPM package is hypothesis #1, not a fallback.** The in-tree `MarkdownPM` package can run a *cached* binary while source is edited ‚ÄĒ so "is my fix even running?" must be answered before debugging the fix. Burned multiple "nothing changed" cycles. **Rule:** on any MarkdownPM rendering bug, change one constant, build, confirm the binary changed first (debug builds: the real code is in `Pommora.debug.dylib`, NOT the 59KB launcher stub). **‚Üí candidate CLAUDE.md quirk.**
- **A scan-logic fix to the index is INERT until the schema version is bumped.** The index is a *versioned regeneratable cache*; `populate` only re-runs when `currentSchemaVersion` ‚Č† the stored version. Both index fixes this session (lenient loader, file-level exclusion) needed a version bump (8‚Üí9‚Üí10) to take effect on an existing DB ‚ÄĒ without it the stale index is reused and the fix appears to do nothing (the exact "no change again" trap). **Rule:** any change to *which files get scanned in* or *how rows are derived* ships WITH a `PommoraIndex.currentSchemaVersion` bump + a one-line comment + the `ConnectionSchemaTests` tripwire updated. **‚Üí candidate CLAUDE.md quirk.**
- **Two discovery contracts must MATCH.** The index scan (`IndexBuilder`) and the UI discovery (`PageContentManager.loadAll`/`descendantFiles`) read pages from disk independently and diverged FOUR ways this session (strict-vs-lenient loader; file-level folder-exclusion; `_`-prefix; non-recursive `children` vs recursive `descendantFiles`). A file visible in the sidebar but absent from the index (or vice versa) is this class of bug. **Rule:** when changing one discovery path, change the other to match; a divergence is a latent bug. (Open instance: deep ‚Č•2-level non-collection subfolders ‚ÄĒ see Pending.)
- **Protocol-default-to-NoOp is invisible.** `pageConnectionResolver` defaults to `NoOpWikiLinkResolver`; a subtree missing injection silently gets the no-op ‚Üí every link muted ‚Üí looks exactly like "index empty." **Rule:** a default-to-no-op injection needs an always-on debug assert at the *injection site* (`NexusEnvironment`), not a probe at the leaf. **‚Üí candidate CLAUDE.md quirk.**
- **Token-accessor contracts must be documented at the type, not the call site.** The placeholder-includes-brackets bug existed because `WikiLinkSelection.placeholder`'s "interior text" contract lived only in a call-site comment. **Rule:** document `displayRange` = full marker span / `placeholder` = interior on the type definition.
- **Index write paths must be integration-tested against a PRE-POPULATED index.** The cascade-wipe (`INSERT OR REPLACE` deleting children) only manifested when re-syncing a type that already had indexed children ‚ÄĒ a state from-scratch tests never create. **Rule:** test upserts against an index that already holds children, asserting they survive ‚ÄĒ not just clean-slate creation.
- **Editor lag is usually per-keystroke work in the styler.** The styler runs on every restyle and called `resolveUniqueEntity` (a synchronous main-thread `dbQueue.read`) once per link, plus 6 `os.Logger` probes fired per keystroke/render with eager string interpolation. Probes were the dominant cost (removed). If lag persists, memoize resolution per restyle-cycle (and a short-lived cross-keystroke cache invalidated on `connectionsChanged`).
